# Basit Hesap Makinesi

Bu küçük Python scripti, kullanıcıdan alınan iki sayıya (veya komut satırı argümanlarıyla verilen iki sayıya) toplama, çıkarma, çarpma ve bölme işlemlerini uygular.

Kullanım:

- Etkileşimli (prompt ile):
```
python calculator.py
```

- Komut satırı argümanları ile:
```
python calculator.py 5 2
```

Örnek (PowerShell):
```
python .\calculator.py 5 2
```

Notlar:
- Bölme işlemi sırasında ikinci sayı `0` ise hata mesajı gösterilir.
- Hem tam sayı hem de ondalık girişleri destekler.
